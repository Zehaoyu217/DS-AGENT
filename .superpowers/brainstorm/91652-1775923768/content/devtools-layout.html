Devtools Tabs

Five views, each purpose-built

1

Events

Real-time stream of everything: tool calls, skill events, sandbox execution, LLM responses, wiki writes, artifact saves. Color-coded by type. Click any event to expand full payload. Filter by type/skill/time range.

2

Skills

Skill registry tree with health indicators. See which functions are loaded, eval pass rates, dependency graph. Quick actions: run evals, reload, check manifest, create new skill. Click any function to see its signature + docs.

3

Config

Live config editor. Change model, sandbox limits, allowed packages, wiki settings — all take effect immediately. Shows diff from defaults. History of config changes this session.

4

Wiki

Browse the LLM wiki. See all pages, their status (confirmed/testing/rejected), cross-references, and the index. Trigger lint. View working.md (what the agent thinks it's doing right now). Read-only — the agent writes, you observe.

5

Evals

Skill eval dashboard. Run evals from the UI, see pass/fail per assertion, view LLM-judged qualitative scores. Compare across iterations. Drill into specific test cases to see agent transcript + grading evidence.
